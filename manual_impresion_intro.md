# Manual de Impresión de Etiquetas

## Introducción

Este manual práctico está diseñado para guiarle paso a paso en el proceso de impresión de etiquetas para órdenes de producción en el sistema WMS. A través de imágenes claras y descriptivas, aprenderá cómo:

- Conectar correctamente la impresora al sistema
- Crear un consecutivo único para cada orden de producción
- Imprimir etiquetas de identificación de manera eficiente

El enfoque visual de este manual le permitirá comprender rápidamente cada proceso sin necesidad de extensas explicaciones escritas. Siga las imágenes en orden secuencial para obtener los mejores resultados en su operación diaria.

Comencemos con la conexión física de su impresora al sistema WMS.

## Requerimientos

1. Tener instalado el servicio de conexión serial, el cual se encarga de comunicar los dispositivos físicos del almacén (Impresoras, Básculas, Etc.) con la aplicación web. Esto es necesario ya que establece el puente de conexión entre la aplicación y los dispositivos físicos.

2. Tener la cuenta de la empresa lista en https://wmsys.fly.dev/
